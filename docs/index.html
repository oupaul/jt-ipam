把 IP 位址管理，留在自己的機房 Keep IP address management in your own rack

jt-ipam 是新世代、可自架的 IPAM——整合 DNS、LibreNMS、OPNsense、AdGuard、Wazuh、Proxmox 與本地 Ollama LLM，並可從 phpIPAM 平滑遷移。資料留在自家，不上雲。 jt-ipam is a modern, self-hosted IPAM — integrating DNS, LibreNMS, OPNsense, AdGuard, Wazuh, Proxmox and a local Ollama LLM, with smooth migration from phpIPAM. Your data stays on-prem.
